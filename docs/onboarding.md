# Onboarding

## Quick Start Checklist
- [ ] Clone repository
- [ ] Install Python 3.11+
- [ ] Create and activate virtual environment
- [ ] Install dependencies from requirements.txt
- [ ] Run az login
- [ ] Run pytest
- [ ] Run streamlit run app/main.py
- [ ] Read docs/outcome.md, docs/requirements.md, docs/architecture.md, docs/nfr.md
- [ ] Read skills and agent files under .github/agents

## Target
A new team member should reach first successful local run in under 60 minutes.
