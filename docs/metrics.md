# Metrics

## Delivery
- Lead time for change
- Deployment frequency
- Change failure rate

## Quality
- Test pass rate
- Defect escape rate
- Mean time to recovery (MTTR)

## Governance
- PRs with docs/context updates (%)
- PRs with QA evidence (%)
- Skills/agents update lag after architecture changes

## AI Governance
- AI-generated PRs reviewed by humans (%)
- AI-generated PRs with test evidence (%)
